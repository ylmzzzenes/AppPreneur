## Tips

- Use Android Studio **View → Tool Windows → Logcat** or emulator **Screenshot** (camera icon).
- Prefer light theme for README consistency, or provide one dark + one light set under `dark/`.
- Target width **1080px** or **390px** (phone); keep file size under **500 KB** each.
- After adding files, verify links in the root [README.md](../../README.md) render on GitHub.
